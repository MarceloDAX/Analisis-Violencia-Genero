# Análisis de Violencia de Género

## Descripción

Proyecto de Ciencia de Datos orientado al análisis de denuncias de violencia de género mediante técnicas de análisis exploratorio de datos y Machine Learning.

El objetivo principal fue identificar patrones asociados a situaciones de mayor riesgo y comprender cómo distintas variables pueden relacionarse con los casos denunciados.

---

## Objetivos

* Analizar la evolución de las denuncias registradas.
* Identificar variables asociadas a situaciones de riesgo elevado.
* Explorar relaciones entre víctimas, agresores y tipos de violencia.
* Aplicar modelos de Machine Learning para la clasificación de casos.

---

## Herramientas utilizadas

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

---

## Metodología

1. Limpieza y preparación de datos.
2. Análisis exploratorio (EDA).
3. Visualización de variables relevantes.
4. Definición de hipótesis.
5. Entrenamiento y evaluación de modelos de Machine Learning.
6. Interpretación de resultados.

---

## Resultados

El proyecto permitió identificar patrones relevantes dentro del conjunto de datos y evaluar modelos predictivos para la clasificación de situaciones de riesgo.
